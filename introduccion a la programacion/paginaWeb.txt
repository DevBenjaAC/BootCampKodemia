Que es una Pagina Web

    HTML -> La estrucutura.
    Css -> Colores, tamaño, animacion.
    Javascript -> Funcionalidades.
    
    Pagina Web
        Documento que vemos en el navegador.
            Puede contener texto, imagenes, videos.

    Sitio Web 
        Coleccion de paginas web.
            Tienen varias paginas interconectadas.

    Aplicación Web
        Sitio web con Funcionalidades
            Se centran en que los usuarios realicen acciones en él.
