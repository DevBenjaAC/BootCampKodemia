SINTAXIS
selector: indica el elemento  o elementos  que seran afectados por la reglamde CSS
Atributos: Representan las caracteristicas que podemos modificar de un elemento o elementos 
Valor: Resultado esperado para cada atributo


====selectores Basicos====
    Selector Universal = * TODOS LOS ELEMENTOS 
    Selector Por tagName = selecciona aquellos elementos que coincidan con el tagname indicado(Nombre de la etiqueta)
    Selector por clase = . selecciona todos los elementos  que contengan el nombre de la clase dentro de su atributo class.
    Selector por id = # (id Asignado) Selecciona aquel unico elemento que contenga el identificador en sua tributo ID
